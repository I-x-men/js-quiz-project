# JavaScript Quiz Project

**Naam:** Jasmin  
**Repository:** js-quiz-project

## Beschrijving
Dit project is een interactieve quiz gemaakt met JavaScript.  
De gebruiker beantwoordt meerkeuzevragen en krijgt direct feedback of een antwoord juist of fout is.

De quiz bevat een spelelement geïnspireerd op platforms zoals **Blooket** en **Kahoot**.  
Een avatar reageert visueel op de antwoorden, wat zorgt voor een speelse en interactieve ervaring.

## Functionaliteiten
- Vragen worden dynamisch op het scherm getoond
- Antwoorden kunnen gekozen worden via knoppen
- Directe feedback bij juiste en foute antwoorden
- Meerdere vragen achter elkaar
- Weergave van de eindscore
- Visuele avatar die reageert op antwoorden

## Gebruikte technieken
